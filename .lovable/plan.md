

# Aditya Machal — Personal Portfolio Website

## Overview
A single-page dark-themed portfolio website with smooth animations, electric blue accents, and responsive design.

## Sections to Build

### 1. Sticky Navbar
- "AM" monogram logo, nav links (About, Skills, Projects, Achievements, Contact)
- Blur + shadow background on scroll, smooth scroll navigation
- Mobile hamburger menu

### 2. Hero Section
- Full viewport, typing animation for "Aditya Machal" with blinking cursor
- Cycling subtitle: ML Engineer → MLOps Enthusiast → LLMOps Builder
- Bio paragraph, two CTA buttons (Download CV + View Projects)
- Animated dot grid background

### 3. Tech Stack Marquee
- Two rows of tech badges scrolling in opposite directions (infinite loop, pause on hover)
- 18 technologies: Python, C++, SQL, Java, PyTorch, TensorFlow, Scikit-learn, FastAPI, Docker, Streamlit, LangChain, CrewAI, MLflow, PostgreSQL, MySQL, Git, Jupyter, Power BI

### 4. Projects Grid
- 2-column grid (1 on mobile), 4 project cards with hover glow effect
- Each card: title, description, tech tags, GitHub link
- Projects: MeetCapsule, Adversarial Robustness MNIST, Process Scheduler Simulator, Global Energy Dashboard

### 5. Achievements & Certifications
- Achievement cards with accent left border and icons
- 5 achievements + 4 certification badges below

### 6. Contact Section
- "Let's Connect" with email, phone, LinkedIn, GitHub as clickable items
- Prominent social icon buttons

### 7. Footer
- Copyright line + social icons

## Design System
- Background: #0a0a0a, Accent: #3B82F6, Text: white
- Inter font, smooth scroll, fade-in-on-scroll animations throughout
- Thin electric blue top border line
